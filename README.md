# Coffee-Shop-Management-System
Coffee Shop Management System is a C# Windows Forms desktop application developed to manage daily operations of a small coffee shop.
The system is designed with role-based access to improve efficiency, accuracy, and workflow between staff members.

This application follows Object-Oriented Programming (OOP) principles and supports basic CRUD operations, file/database storage, and user-friendly interfaces.
